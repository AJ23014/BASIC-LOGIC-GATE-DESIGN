# Basic Logic Gate Design (Verilog HDL)
## OverviewThis project implements standard digital logic gates using Verilog HDL. It demonstrates fundamental combinational logic modeling, simulation verification, and synthesizable RTL design methodologies using Xilinx Vivado.
---
## Features* **Complete Gate Library:** Implementation of core logic gates.* **Structural & Behavioral Modeling:** Clean, synthesizable Verilog code styles.* **Functional Testbench:** Setup for timing simulation and truth table validation.
